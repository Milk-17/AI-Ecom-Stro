วิธีใช้
1.ลากไฟล์ .env มาใว้ที่โฟลเดอร์ server
2.สร้าง terminal ใหม่ และ cd server แล้วรัน npm i กับ npm i nodemon
3.สร้าง terminal ใหม่ และ cd client แล้วรัน npm i
4.สร้าง terminal ใหม่ และ cd server และรัน npx prisma migrate dev
5.สร้าง terminal ใหม่ และ cd server แล้วรัน npm start
6.สร้าง terminal ใหม่ และ cd clien แล้วรัน npm run dev
